# 项目上下文

## 仓库与技术栈

- 当前仓库路径为 `d:\Project\anti`，来源于用户 fork 的 `Youkies/Ant-Browser`。
- 项目是桌面应用，技术栈为 `Wails + Go + React + TypeScript + Vite + Tailwind`。
- 当前本地开发环境已可正常安装依赖、构建并启动。

## 本地约定

- `.ai_memory/` 作为仓库本地记忆库，当前允许纳入版本控制。
- `config.yaml` 属于本地运行态配置文件，之前多次约定默认不随代码提交，除非用户明确要求。
- 本机 `Wails CLI` 曾将项目依赖对齐到 `v2.12.0`，因此仓库里存在相应的 `go.mod` / `go.sum` 变化。

## 已完成的稳定改动

### 代理池

- 代理池页面已做多轮布局收口，适配小窗使用场景。
- 操作列已改为图标按钮，并优化了表格信息密度。
- 节点支持本地 SVG 国旗展示，国家信息优先在节点名和 IP 健康区域复用。
- 已补充 `AnyTLS` 节点兼容与对应测试。

### 实例列表

- 实例列表已针对小窗做多轮压缩与布局修正。
- `核心`、`快捷打开码`、`上次更新` 等低频信息已收进详情。
- 代理列可复用代理 IP 信息展示国旗。
- 实例列表中的“扩容实例”入口与扩容弹窗已删除。

### 容量与扩容系统

- 项目已移除实例数量容量限制，创建和复制实例不再受 `max_profile_limit` 之类额度控制。
- 控制台中的扩容系统区块、兑换码输入、GitHub Star 赠送入口已删除。
- 隐藏管理员兑换码生成页 `/admin/keygen` 已移除。
- 后端已删除兑换码、额度状态同步及对应容量限制逻辑；旧配置中的相关字段即使仍存在，也不再参与运行逻辑。

### 链式代理

- 当前已实现单层前置节点能力，字段名为 `preProxyId`。
- `本地代理` 允许作为前置节点。
- 前置节点只在代理池中配置，实例列表选择目标节点时自动复用代理池设置，无需额外操作。
- 代理池主表会显示 `前置节点：节点名`。
- 实例启动、测速、IP 健康已接入统一的链式代理解析与桥接逻辑。

### 路由

- Wails 桌面端前端路由已切换为 `HashRouter`。
- 原因是当前 `wails.localhost` 静态服务没有 SPA fallback，直接访问 `/browser/edit/:id` 这类子路由会返回 `404`。

### 主题与标题栏

- 主题系统已统一为 `system / light / dark` 三档，不再依赖旧的多套自定义配色主题作为主入口。
- 前端主题状态会同步到 `app_settings.theme` 与 `app-theme`，设置页和全局界面共用同一套主题来源。
- Wails Windows 原生标题栏已接入主题联动：浅色、深色和跟随系统都会同步影响窗口标题栏与窗口背景色。
- 当前已进一步切换为 Wails `Frameless` 无边框窗口，应用顶部区域开始接管自定义标题栏职责。
- 自定义标题栏复用了现有 `Sidebar` 顶部 Logo 区与 `Topbar`，支持拖拽、双击最大化、最小化、最大化/还原和关闭按钮。
- 顶部标题栏中的原搜索框已改造成全局快速启动入口，点击后会打开现有 `QuickLaunchModal`，而不是保留一个无实际作用的输入框。
- 标题栏左侧的占位文案已替换为当前页面标题，并支持静态路由与实例详情/编辑/复制等动态页面标题映射。
- 顶部快速启动入口已收成单行截断展示，避免标题栏窄宽度下出现换行。
- 应用当前 Logo 与窗口图标资源均已替换为 `Youkies_Browser_icon_202603311631-removebg-preview (1).png`。
